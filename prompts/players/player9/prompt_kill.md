### 首夜刀人策略

**目标选择公式**：  
`首刀目标 = argmax( 发言攻击性 × 0.7 + 逻辑深度 × 0.3 )`

**具体优先级**：

1. **高逻辑玩家**（发言逻辑系数>0.8）
   - 特征：首轮发言结构清晰，善于提问
   - 刀法价值：消除潜在威胁

2. **话术密度高玩家**（发言字数>平均30%）
   - 特征：分析全面，善于整合信息
   - 刀法价值：防止信息整合

3. **回避高风险目标**：
   - 绝对不刀划水玩家（易暴露狼队刀法逻辑）
   - 避免刀疑似女巫（防首夜自救）

### 中期刀人策略（第2-3夜）

**刀法决策矩阵**：

| 情况                | 刀人目标                     |
|---------------------|----------------------------|
| 预言家未明          | 刀信息整合者                |
| 预言家已明          | 刀预言家+女巫解药已用       |
| 猎人已暴露          | 刀平民/女巫（留猎人）        |
| 出现银水（女巫救）  | 刀银水玩家                  |

**核心技巧**：

1. **断链刀法**：  
   刀死能验证他人发言的玩家（如最后发言者）

2. **嫁祸刀法**：  
   刀与某玩家有激烈矛盾的对手，嫁祸给该玩家

3. **信息污染刀**：  
   故意刀发言混乱玩家，制造"狼队不会刀TA"的假象

### 终局刀人策略（第4夜+）

**决胜公式**：  
`刀人目标 = argmax( 身份威胁值 ÷ 可疑度 )`

**身份威胁值计算**：

1. 预言家 = 1.2 × (存活天数)
2. 女巫 = 1.0 × (剩余药数)
3. 猎人 = 0.8 × (开枪威慑力)
4. 平民 = 0.3 × (逻辑能力)

**终局三原则**：

1. **平衡原则**：  
   保持`狼人数量 ≥ 好人数量 - 1`

   ```python
   if 存活好人 == 狼人数量+1:
       刀神职 > 刀平民
   ```

2. **猎人陷阱**：
当存活玩家≤4时，留猎人刀平民，诱导猎人开枪误杀好人

3. **神职清除序列**：
女巫 > 预言家 > 猎人（按威胁度排序）
