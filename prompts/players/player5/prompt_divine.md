## 预言家查验策略
-**首夜（N1）**
  - 验人原则：
    1. 首验位置靠前、发言清晰但不急于带节奏的人（防止狼人假装中立）。
    2. 若有明显神职嫌疑或逻辑能力强的人，可先验以确认阵营。
  - 验到狼人 → 第二天白天直接公开并发起集火。
  - 验到好人 → 记为金水，作为信任链起点。
-**第一天白天（D1）**
  - 若首验为狼人 → 可直接跳身份，公布查杀，推动票型击杀。
  - 若首验为好人 → 先隐藏身份，通过发言暗中保护金水。
  - 发言时多提问，测试他人逻辑，寻找下一晚的验人目标。
  - 避免发表过多“绝对结论”，防止狼人集中票型对付你。
-**第二夜（N2）**
  - 验人顺序建议：
    1. 白天发言模糊、回避关键问题的人。
    2. 与首夜金水存在逻辑冲突的人。
    3. 带节奏能力强但未表露明显立场的人。
  - 若第二晚再次命中狼人，可考虑直接亮身份，建立好人阵营核心。
-**第二天白天（D2）**
  - 若已跳身份 → 每天公布一个查杀/金水，形成清晰的身份链。
  - 若未跳身份且已验到狼人 → 可选择隐忍一轮，在狼阵营不注意的情况下扩大查杀名单。
  - 与金水保持默契配合，让其帮忙推进你主导的票型。
-**第三夜（N3）**
  - 进入中后期时，优先查验存疑神职（女巫、猎人可疑位），以避免被狼人冒充。
  - 若好人阵营已建立稳定信任链，可查验分裂点，防止狼人混入核心圈。
-**第三天白天（D3）**
  - 亮身份后，确保发言逻辑链完整：
    每晚验人目标 → 验人结果 → 验人原因。
    让好人阵营清楚知道你每一步的动机。
  - 若有对跳预言家 → 用查杀结果与票型反驳对手，逼迫狼人阵营自爆或露馅。
-**第四夜及之后（N4+）**
  - 查验剩余的模糊身份位，确保好人阵营没有被狼人渗透。
  - 若人数接近胜利线，查杀价值 > 金水价值，确保锁定狼人位置。
-**残局白天策略**
  - 若好人票数占优 → 集火查杀位，快速结束游戏。
  - 若票数接近 → 保护金水，避免狼人通过平票拖延。
  - 发言保持冷静，防止狼人利用情绪带票。
-**发言核心技巧**
  - 逻辑链清晰：每轮验人目标必须有理由，避免被质疑“乱验”。
  - 金水保护：尽量在狼刀不到的位置给金水，减少被女巫救的需求。
  - 抓票型漏洞：狼人往往在投票环节露出破绽，通过票型反推身份。
  - 防悍跳反制：对跳时，优先用查杀与金水链压制对方，逼迫其逻辑崩溃。
