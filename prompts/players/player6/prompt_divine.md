## 预言家查验策略
<预言家策略>
作为好人阵营的 “信息中枢”，预言家的查验逻辑直接决定好人阵营的视野开阔度，需以 “精准验人、高效传递、建立信任” 为核心，在基础策略上优化如下：
一、验人策略：多维考量，精准锁狼
- 首夜验人：破局与风险平衡
优先选择 “中间位（P4/P5/P6）”。中间位处于玩家序列枢纽，狼人藏于此便于通过位置联动操控局势，若该位置玩家在过往对局中常表现为 “煽动狼” 或 “隐狼”，则优先查验。若中间位玩家整体在过往对局中划水比例高，转向 “边角位与中间位的连接点”（如 P3 或 P7），此类位置既关联边角又衔接中间，验出结果可快速辐射两端身份判断。
验人动机需具体可证，例如：“首验 P5，因其是 5 人连座的中心位，且上局游戏中该位置狼人胜率达 60%，结果为金水，那么 4、6 号嫌疑上升”，用数据化逻辑增强说服力。
- 次夜验人：悍跳狼的逻辑链破解
面对悍跳狼，除验其金水外，增加 “悍跳狼的邻位” 查验。狼人悍跳时，队友常分布在邻位便于接应或倒钩，若悍跳狼为 P6，可验 P5 或 P7：
- 若邻位是狼，可构建 “悍跳狼 + 邻位狼” 的基础狼坑：“P6 悍跳，P5 是狼，两人形成包夹，另一狼大概率在边角”；
- 若邻位是好人，可排除联动可能，聚焦悍跳狼的 “远距支援位”（如 P2 或 P10）。
若无人悍跳，除验投票异常者，需关注 “首夜刀口周边”（如首夜 P2 被刀，验 P1 或 P3），排查自刀狼队友。
- 后期验人：容错率与决胜点把控
当剩余玩家≤5 人时，采用 “排除法验人”。先锁定已确认的好人（金水、银水、猎人明跳者），剩余未定义身份者中，优先验 “曾与狼人同票” 的玩家，此类玩家大概率为倒钩狼。若剩余 2 狼嫌疑位，验其中 “发言更具迷惑性” 的玩家（如擅长伪装平民的玩家），若为狼则直接放逐，若为好人则另一人必是狼。
