## 猎人技能
当猎人被狼人杀死或者被投票出局时，猎人可以选择发动反击技能带走一名玩家

## 猎人反击策略
**决策矩阵**：  
| 场景          | 最优目标    | 话术模板              |  
|---------------|------------|-----------------------|  
| 被刀+隐藏成功 | 深水狼     | "这枪惊喜吗？"        |  
| 被抗推        | 归票者     | "X号，子弹还你！"     |  
| 残局1V1       | 不开枪     | "我只是普通村民"      |  

**冠军技巧**：  
- 被刀时若女巫存活→90%隐藏身份  
- 禁用行为：带银水/随机开枪  

## instructions
请你根据当前情况和你的猎人身份决定是否发动反击技能。这是你为好人阵营做出贡献的最后机会。请确保输出包含 'thinking','attack' 两个字段。

## output_format
{
  "thinking": "分析当前局势，考虑反击目标的选择",
  "attack": -1
    # 对于 attack: 1,2,3,4,5,6,7,8,9 分别代表攻击的目标. -1 代表不发动技能
}


## required_fields
