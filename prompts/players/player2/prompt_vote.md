## 狼人杀女巫投票策略深度解析

### 一、投票的基本原则

#### 1. 女巫投票的特殊性
**核心地位：**
- 女巫的票往往具有导向性
- 需要在隐藏身份和发挥作用间平衡
- 投票行为会被重点分析

**三大原则：**
1. **一致性原则**：投票逻辑要与发言保持一致
2. **价值最大化**：每一票都要为好人阵营创造价值
3. **信息传递**：通过投票向好人传递暗号

### 二、首轮投票策略

#### 1. 第一天投票（信息匮乏期）
**基本策略：**
- **跟随真预言家**：如果确认真预言家，坚决跟票
- **避免乱投**：宁可弃票也不乱投好人
- **观察投票型**：记录每个人的投票，为后续分析做准备

**投票选择优先级：**
```
1. 预言家的查杀位（最优先）
2. 发言有明显狼人爆点的玩家
3. 对跳预言家中可疑度更高的
4. 划水严重、发言毫无营养的
5. 弃票（实在无法判断时）
```

**示例场景：**
```
场景：两个预言家对跳，你倾向于相信3号
投票策略：
- 如果3号查杀了5号 → 投5号
- 如果3号没有查杀 → 投对跳的预言家
- 如果都不确定 → 可以投划水位或弃票
```

#### 2. 首轮投票的信息收集
**重点观察：**
- **抱团投票**：哪些人投票一致（可能是狼队）
- **分票情况**：谁在关键时刻改票
- **弃票玩家**：谁选择了观望

**记录模板：**
```
第一轮投票情况：
- 投X号：1、3、5、7（可能有狼队配合）
- 投Y号：2、4、6、8（可能有好人被误导）
- 弃票：9、10、11、12（身份不明，需继续观察）
```

### 三、关键轮次投票策略

#### 1. 平票局面（如3-3、4-4）
**女巫的关键一票：**
```
判断标准：
1. 两个PK对象谁的好人面更大
2. 谁活着对好人阵营更有利
3. 谁的发言更有逻辑性
4. 综合之前的投票记录

投票原则：
- 宁错杀，勿放过（后期）
- 宁放过，勿错杀（前期）
- 保神职，出平民（中期）
```

**案例分析：**
```
场景：第三天，3号和7号PK，票数3:3
分析：
- 3号：自称平民，发言逻辑清晰，但投票记录可疑
- 7号：暗示有身份，发言一般，但一直跟对票
决策：投3号（保身份，即使错了损失较小）
```

#### 2. 归票阶段
**后置位女巫的优势：**
- 可以看清票型走向
- 能够决定最终结果
- 适时暴露身份带队

**归票策略：**
```
情况一：票型明确，跟票即可
"前面都投X号了，我也跟票，统一出X。"

情况二：票型分散，需要引导
"我建议统一投X号，理由已经很充分了。"

情况三：需要改变局面
"等等，我觉得不该出X号，应该出Y号！"
[必要时亮身份] "相信我，我是女巫！"
```

### 四、基于不同因素的投票决策

#### 1. 基于发言的投票
**发言分析要点：**
```
强烈投出（狼人特征）：
✓ 逻辑混乱，前后矛盾
✓ 带节奏，煽动情绪
✓ 站边错误，保狼踩好人
✓ 划水摸鱼，毫无观点

谨慎投出（可能是好人）：
- 逻辑清晰但站边错误（可能被误导）
- 情绪激动但真情流露（可能是真的急了）
- 新手表现，发言稚嫩（可能真的不会玩）
```

#### 2. 基于身份嫌疑度的投票
**嫌疑度评估体系：**
```
高嫌疑度（8-10分）→ 坚决投出
- 被查杀的玩家（10分）
- 对跳失败的假身份（9分）
- 多次站错边+投票可疑（8分）

中嫌疑度（5-7分）→ 视情况投
- 发言有漏洞但不致命（7分）
- 投票记录可疑但发言尚可（6分）
- 划水但没有明显狼人行为（5分）

低嫌疑度（1-4分）→ 一般不投
- 被发金水的玩家（1分）
- 逻辑清晰的好人牌（2分）
- 已验证的真身份（1分）
```

#### 3. 基于场上局势的投票
**局势判断与投票策略：**

**好人领先局面：**
```
特点：狼人已暴露2-3个，局势明朗
投票策略：
- 稳扎稳打，优先出铁狼
- 不冒险，宁可出错也不给狼队机会
- 保护关键好人，特别是预言家
```

**势均力敌局面：**
```
特点：双方人数接近，每一票都关键
投票策略：
- 谨慎投票，避免误推好人
- 必要时暴露身份掌控局面
- 优先投出煽动能力强的狼人
```

**狼人领先局面：**
```
特点：好人人数劣势，需要奇迹
投票策略：
- 激进投票，宁错杀不放过
- 集中火力，避免分票
- 相信直觉，果断出手
```

```
- 谁激烈反对（可能是狼同伴）
- 谁积极附和（可能是踩狼队友）
```

#### 3. 威慑性投票

