## 预言家查验策略
1. **首夜查验逻辑**  
   - 随机选择中位玩家（4-6号位）查验，避免首验高调/低调玩家  
   - *高级玩法*：若玩家发言存在逻辑断层（如前后矛盾），优先查验；或故意验疑似狼人（测试其反应链）  

2. **后续查验优先级**  
   - Tier1目标：投票异常者（如盲跟票）、攻击神职者  
   - Tier2目标：划水玩家（发言<20字）、过度辩护者  
   - *高级玩法*：  
     • 逆向思维：故意验"像好人的玩家"（破狼人倒钩）  
     • 查验链：连续验同一阵营玩家（快速清坑）  

3. **信息管理**  
   - 跳身份时机：  
     ✓ 首验出狼 → 次日立即跳（带节奏）  
     ✓ 验出关键神职（女巫/猎人） → 隐藏身份  
   - 发言技巧（≤340字）：  
     • 模板："昨夜验[X]，身份[结果]。建议焦点[Y]（可疑者）"  
     • *高级玩法*：假报查验（谎称验A是狼，实则验B）引狼暴露  

4. **风险管理**  
   - 防暴露：避免固定发言模式（如总在末位归票）  
   - 防针对：被怀疑时优先自证（例："若我是狼，为何报[玩家]好人？"）  
   - 轮次意识：第3夜后停止验民，专注找深水狼  

5. **决策树**  
   if 首夜验出狼人 → 跳身份带队  
   elif 女巫未暴露 → 次夜验疑似女巫（建信任链）  
   else → 查验票型异常者（投票/发言背离度>60%）
