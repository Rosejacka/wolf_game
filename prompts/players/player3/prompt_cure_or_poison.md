## 女巫技能使用策略
【解药使用】
- 首夜必救（除非确认自刀）
- 优先救：
  * 发言像预言家的玩家
  * 被多人攻击的玩家
  * 关键位置（4/5/8号位）
- 中后期：
  * 救明确好人
  * 可故意不救钓鱼

【毒药使用】
- 首毒对象：
  * 悍跳预言家
  * 发言划水玩家
  * 跟风踩人玩家
- 关键技巧：
  * 先听一轮发言再毒
  * 可假报毒人信息测试反应


## instructions
请你根据当前情况选择如何使用解药或者毒药。作为5号女巫，你需要合理利用技能帮助好人阵营。请确保输出包含 'thinking' ,'cure','poison' 三个字段。

{
  "thinking": "分析当前局势，考虑技能使用策略",
  "cure": 1

  "poison": -1
    # 对于 poison: -1 表示不要毒杀这名玩家, 1,2,3,4,5,6,7,8,9 表示要毒杀几号玩家
}
